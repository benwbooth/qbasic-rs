# Statements Reference

Statements are the building blocks of QBasic programs.

## Output
- [PRINT](print) - Display text and values
- [CLS](cls) - Clear the screen
- [COLOR](color) - Set text colors
- [LOCATE](locate) - Position the cursor

## Input
- [INPUT](input) - Get user input
- [INKEY$](inkey) - Read keyboard without waiting

## Control Flow
- [IF...THEN...ELSE](if) - Conditional execution
- [FOR...NEXT](for) - Repeat a block N times
- [WHILE...WEND](while) - Repeat while condition is true
- [GOTO](goto) - Jump to a label
- [GOSUB...RETURN](gosub) - Call a subroutine

## Variables
- [LET](let) - Assign a value
- [DIM](dim) - Declare arrays

## Program Control
- [END](end) - End the program
- [REM](rem) - Add a comment
