# REM Statement

Adds a comment to your program.

## Syntax
```
REM comment text
' comment text
```

## Example
```
REM This is a comment
' This is also a comment
x = 10  ' Inline comment
```

## Notes
- Comments are ignored during execution
- Use ' (apostrophe) for shorter syntax
- Comments can appear at end of lines
- Good comments explain "why", not "what"
