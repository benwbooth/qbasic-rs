# CLS Statement

Clears the screen and moves cursor to top-left.

## Syntax
```
CLS
```

## Example
```
CLS
PRINT "Fresh start!"
```

## Notes
- Clears all text from the screen
- Resets cursor to row 1, column 1
- Does not change the current color settings
