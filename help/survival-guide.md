# QBasic Survival Guide

A quick reference for the most essential commands and shortcuts.

## Getting Around

| Action | Key |
|--------|-----|
| Access menus | Alt + highlighted letter |
| Help on any item | F1 |
| Switch to Immediate window | F6 |
| Search/Find | Ctrl+F |

## Editing Keys

| Action | Key |
|--------|-----|
| Move to start/end of line | Home / End |
| Move to start/end of program | Ctrl+Home / Ctrl+End |
| Delete current line | Ctrl+Y |
| Copy selected text | Ctrl+C |
| Paste | Ctrl+V |
| Cut | Ctrl+X |
| Undo | Ctrl+Z |
| Select all | Ctrl+A |

## Running Programs

| Action | Key |
|--------|-----|
| Run program | [F5](shortcuts) |
| Restart program | Shift+F5 |
| Step through code | [F8](shortcuts) |
| Set/clear breakpoint | [F9](shortcuts) |
| Stop running program | Ctrl+Break or Ctrl+C |

## File Operations

| Action | Key |
|--------|-----|
| New program | Ctrl+N |
| Open file | Ctrl+O |
| Save file | Ctrl+S |
| Save As | Ctrl+Shift+S |
| Exit QBasic | Alt+F, X |

## Getting Help

| Action | Key |
|--------|-----|
| Help on current word | F1 |
| Help Index | Shift+F1 |
| Help Contents | Alt+H, C |
