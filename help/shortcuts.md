# Keyboard Shortcuts

Complete reference of all keyboard shortcuts.

## Navigation

| Key | Action |
|-----|--------|
| Arrow keys | Move cursor |
| Home | Move to beginning of line |
| End | Move to end of line |
| Ctrl+Home | Move to beginning of program |
| Ctrl+End | Move to end of program |
| Page Up | Scroll up one page |
| Page Down | Scroll down one page |
| Ctrl+G | Go to line number |

## Selection

| Key | Action |
|-----|--------|
| Shift+Arrow | Extend selection |
| Shift+Home | Select to beginning of line |
| Shift+End | Select to end of line |
| Ctrl+A | Select all |
| Double-click | Select word |
| Triple-click | Select line |

## Editing

| Key | Action |
|-----|--------|
| Ctrl+C | Copy selection |
| Ctrl+X | Cut selection |
| Ctrl+V | Paste |
| Ctrl+Z | Undo |
| Ctrl+Y | Delete line |
| Delete | Delete character |
| Backspace | Delete previous character |
| Insert | Toggle insert/overwrite mode |

## Search

| Key | Action |
|-----|--------|
| Ctrl+F | Find |
| F3 | Find next |
| Ctrl+H | Find and replace |

## Running and Debugging

| Key | Action |
|-----|--------|
| F5 | Run program |
| Shift+F5 | Restart program |
| F8 | Step (execute one line) |
| F9 | Toggle breakpoint |
| Ctrl+Break | Stop program |

## File Operations

| Key | Action |
|-----|--------|
| Ctrl+N | New program |
| Ctrl+O | Open file |
| Ctrl+S | Save file |
| Ctrl+Shift+S | Save As |
| Ctrl+P | Print |

## Windows and Menus

| Key | Action |
|-----|--------|
| F6 | Switch between Editor and Immediate |
| Alt | Activate menu bar |
| Alt+F | File menu |
| Alt+E | Edit menu |
| Alt+V | View menu |
| Alt+S | Search menu |
| Alt+R | Run menu |
| Alt+D | Debug menu |
| Alt+O | Options menu |
| Alt+H | Help menu |
| F10 | Activate menu bar |
| Escape | Close menu/dialog |

## Help

| Key | Action |
|-----|--------|
| F1 | Help on current item |
| Shift+F1 | Help index |
