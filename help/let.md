# LET Statement

Assigns a value to a variable.

## Syntax
```
[LET] variable = expression
```

## Examples
```
LET x = 10           ' Explicit LET
y = 20               ' LET is optional
name$ = "Alice"      ' String variable
total = x + y        ' Expression
```

## Notes
- LET keyword is optional
- Variables are created on first use
- Use $ suffix for string variables
- Use % suffix for integer variables
