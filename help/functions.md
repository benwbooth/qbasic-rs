# Functions Reference

Functions return values and can be used in expressions.

## Math Functions
- [ABS](abs) - Absolute value
- [INT](int) - Integer part (truncate)
- [RND](rnd) - Random number (0 to 1)

## String Functions
- [LEN](len) - String length
- [LEFT$](left) - Left portion of string
- [RIGHT$](right) - Right portion of string
- [MID$](mid) - Middle portion of string
- [CHR$](chr) - Character from ASCII code
- [ASC](asc) - ASCII code from character
- [STR$](str) - Convert number to string
- [VAL](val) - Convert string to number

## System Functions
- [TIMER](timer) - Seconds since midnight
- [INKEY$](inkey) - Read keyboard
