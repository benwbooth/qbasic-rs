# END Statement

Terminates program execution.

## Syntax
```
END
```

## Example
```
PRINT "Processing..."
IF error THEN
    PRINT "Error occurred!"
    END
END IF
PRINT "Done!"
END
```

## Notes
- Stops all program execution
- Can be placed anywhere in the program
- Multiple END statements are allowed
- Program also ends when it runs past the last line
