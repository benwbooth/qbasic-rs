# Using This Help System

How to navigate and use the QBasic help system.

## Getting Help

| Action | Key |
|--------|-----|
| Help on word under cursor | F1 |
| Help Index | Shift+F1 |
| Help Contents | Alt+H, C |

## Navigating Help

### Hyperlinks

Words shown in **[brackets]** are links to other topics.
Click on them or use Tab to move between links, then press Enter.

### Scrolling

| Key | Action |
|-----|--------|
| Up/Down arrows | Scroll one line |
| Page Up/Page Down | Scroll one page |
| Home | Go to top |
| End | Go to bottom |

### Navigation

| Key | Action |
|-----|--------|
| Tab | Move to next link |
| Shift+Tab | Move to previous link |
| Enter | Follow link |
| Backspace | Go back to previous topic |
| Escape | Close help |

## Context-Sensitive Help

Press F1 while the cursor is on any keyword to get help for that specific command. For example:

1. Type `PRINT` in your program
2. Position cursor on the word PRINT
3. Press F1

The help for the PRINT statement will appear.

## Menu Help

Press F1 while a menu is open to get help about that menu's commands.
