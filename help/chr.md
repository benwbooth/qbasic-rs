# CHR$ Function

Returns the character for an ASCII code.

## Syntax
```
CHR$(code)
```

## Example
```
PRINT CHR$(65)   ' Prints A
PRINT CHR$(13)   ' Carriage return
newline$ = CHR$(10)
```

## Common Codes
| Code | Character |
|------|-----------|
| 13 | Carriage return |
| 10 | Line feed |
| 32 | Space |
| 65-90 | A-Z |
| 97-122 | a-z |
